    SAM 2: 画像と動画のセグメント化

    MetaのAI、FAIR

    Nikhila Ravi, Valentin Gabeur, Yuan-Ting Hu, Ronghang Hu, Chaitanya Ryali, Tengyu Ma, Haitham Khedr, Roman Rädle, Chloe Rolland, Laura Gustafson, Eric Mintun, Junting Pan, Kalyan Vasudev Alwala, Nicolas Carion, Chao-Yuan Wu, Ross Girshick, Piotr Dollár, Christoph Feichtenhofer

    [論文] [プロジェクト] [デモ] [データセット] [ブログ] [BibTeX]

    セグメント化モデル2 (SAM 2)は、画像と動画におけるプロンプト可能な視覚セグメンテーションを解決するための基盤モデルです。画像を単一フレームの動画として扱うことで、SAMを動画に拡張します。モデル設計は、リアルタイム動画処理のためのストリーミングメモリを備えたシンプルなトランスフォーマーアーキテクチャです。ユーザーとの対話を通じてモデルとデータを改善するモデルインザループデータエンジンを構築し、SA-Vデータセットを収集しました。これは、これまでで最大の動画セグメンテーションデータセットです。私たちのデータでトレーニングされたSAM 2は、幅広いタスクと視覚領域で強力なパフォーマンスを提供します。

    最新の更新情報

    2024年9月30日 -- SAM 2.1 開発者スイート (新しいチェックポイント、トレーニングコード、ウェブデモ) がリリースされました

    • 改良されたモデルチェックポイントの新しいスイート (SAM 2.1) がリリースされました。詳細はモデル説明をご覧ください。
      • 新しいSAM 2.1チェックポイントを使用するには、このリポジトリの最新のモデルコードが必要です。以前のバージョンをインストールしている場合は、まずpip uninstall SAM-2を実行し、このリポジトリから最新のコードを取得してください (git pull)。その後、以下のインストール手順に従ってリポジトリを再インストールしてください。
    • トレーニング (および微調整) コードがリリースされました。開始方法についてはtraining/README.mdをご覧ください。
    • SAM 2ウェブデモのフロントエンド＋バックエンドコードがリリースされました。詳細はdemo/README.mdをご覧ください。

    インストール

    SAM 2を使用する前にインストールする必要があります。このコードはpython>=3.10が必要です。そしてtorchvision>=0.18.1です。以下の手順に従ってくださいこちらでPyTorchとTorchVisionの依存関係をインストールします。SAM 2をGPUマシンにインストールするには、次のコマンドを使用します:

    git clone https://github.com/facebookresearch/sam2.git && cd sam2

pip install -e .

    Windowsにインストールする場合は、Windows Subsystem for Linux (WSL)とUbuntuを使用することを強くお勧めします。

    SAM 2の予測機能を使用し、例のノートブックを実行するには、jupyterおよびmatplotlibが必要であり、以下のコマンドでインストールできます:

    pip install -e ".[notebooks]"

    注意:

    1. 新しいPython環境をAnacondaを使用して作成し、このインストールのためにPyTorch 2.3.1（またはそれ以上）をpipを使用してインストールすることをお勧めします。詳細はhttps://pytorch.org/をご覧ください。現在の環境にPyTorchのバージョンが2.3.1未満の場合、上記のインストールコマンドは最新のPyTorchバージョンにアップグレードしようとします。pip.
    2. 上記の手順では、nvccコンパイラを使用してカスタムCUDAカーネルをコンパイルする必要があります。マシンにまだインストールされていない場合は、CUDAツールキットをインストールしてください。PyTorch CUDAバージョンに一致するバージョンを選択してください。
    3. インストール中にSAM 2 CUDA拡張のビルドに失敗しましたというメッセージが表示された場合でも無視してSAM 2を使用できます（一部の後処理機能が制限される場合がありますが、ほとんどの場合結果には影響しません）。

    詳細についてはINSTALL.mdをご覧ください。

    開始方法

    チェックポイントのダウンロード

    まず、モデルのチェックポイントをダウンロードする必要があります。すべてのモデルチェックポイントは以下のコマンドでダウンロードできます:

    cd checkpoints && \
./download_ckpts.sh && \
cd ..

    または個別に以下から:

    • sam2.1_hiera_tiny.pt
    • sam2.1_hiera_small.pt
    • sam2.1_hiera_base_plus.pt
    • sam2.1_hiera_large.pt

    （これらはSAM 2.1として示される改良されたチェックポイントです。詳細はモデル説明をご覧ください。）

    次に、SAM 2を使用して画像および動画の予測を行うことができます。

    画像予測

    SAM 2は静止画像においてSAMのすべての機能を備えており、画像用途のためにSAMに近い予測APIを提供します。SAM2ImagePredictorクラスは画像プロンプトのための簡単なインターフェースを提供します。

    import torch
from sam2.build_sam import build_sam2
from sam2.sam2_image_predictor import SAM2ImagePredictor

checkpoint = "./checkpoints/sam2.1_hiera_large.pt"
model_cfg = "configs/sam2.1/sam2.1_hiera_l.yaml"
predictor = SAM2ImagePredictor(build_sam2(model_cfg, checkpoint))

with torch.inference_mode(), torch.autocast("cuda", dtype=torch.bfloat16):
    predictor.set_image(<your_image>)
    masks, _, _ = predictor.predict(<input_prompts>)

    例についてはimage_predictor_example.ipynb（Colabでもこちら）をご覧ください。

    SAM 2はまた、SAMと同様に画像上での自動マスク生成をサポートしています。詳細はautomatic_mask_generator_example.ipynb（Colabでもこちら）をご覧ください。

    動画予測

    動画におけるプロンプト可能なセグメンテーションとトラッキングのために、プロンプトを追加し、マスクレットを動画全体に伝播させるAPIを備えた動画予測機能を提供します。SAM 2は複数のオブジェクトに対する動画推論をサポートし、各動画内の相互作用を追跡する推論状態を使用します。

    import torch
from sam2.build_sam import build_sam2_video_predictor

checkpoint = "./checkpoints/sam2.1_hiera_large.pt"
model_cfg = "configs/sam2.1/sam2.1_hiera_l.yaml"
predictor = build_sam2_video_predictor(model_cfg, checkpoint)

with torch.inference_mode(), torch.autocast("cuda", dtype=torch.bfloat16):
    state = predictor.init_state(<your_video>)

    # add new prompts and instantly get the output on the same frame
    frame_idx, object_ids, masks = predictor.add_new_points_or_box(state, <your_prompts>):

    # propagate the prompts to get masklets throughout the video
    for frame_idx, object_ids, masks in predictor.propagate_in_video(state):
        ...

    詳細についてはvideo_predictor_example.ipynb（Colabでもこちら）をご覧ください。

    🤗 Hugging Faceからのロード

    また、モデルはHugging Faceからもロードできます（pip install huggingface_hub).

    画像予測の場合:

    import torch
from sam2.sam2_image_predictor import SAM2ImagePredictor

predictor = SAM2ImagePredictor.from_pretrained("facebook/sam2-hiera-large")

with torch.inference_mode(), torch.autocast("cuda", dtype=torch.bfloat16):
    predictor.set_image(<your_image>)
    masks, _, _ = predictor.predict(<input_prompts>)
